Describe objects and relations

This is a browser based demo for automatically generating descrptions for objects and their relations using neural language models. You need to choose the picture and two objects, then system will generate word sequences based on object1 in relation with object2.


Requirements

You need Google Chrome browser to be able to run the demo. The models are around 60 MB.

How to use it

1. Choose a photo.
2. Draw bounding box for object1.
3. Draw bounding box for object2.
4. Repeat 2 and 3 until you are happy with bounding boxes.
5. Click on Generate button.

About the demo

The model is built in Keras then ported for Tensorflow.js. The conditional neural language model is trained on 100K images from VisualGenome's relatinoships dataset with around 15 descrptions per image. For convenience of this online demo, we used a pretrained mobilenet to extract visual features. In addition to convolutional features, the bouning box arrangements are encoded as a vector of spatial features.
